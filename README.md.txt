# Analizador de Datos Pokémon

Proyecto final de Ciencia de Datos

## Descripción
Aplicación desarrollada en Python para analizar datos de Pokémon desde un archivo Excel.

Funciones:

- Filtrar Pokémon por tipo
- Generar gráficas dinámicas
- Exportar reportes a Excel
- Visualizar estadísticas

## Librerías usadas

- pandas
- matplotlib
- openpyxl
- customtkinter

## Ejecución

Instalar dependencias:

pip install -r requirements.txt

Ejecutar:

python main.py
